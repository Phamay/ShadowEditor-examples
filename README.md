# ShadowEditor示例

为了避免ShadowEditor项目由于添加大量材质和模型过大，所以新建个项目放例子。

所有示例和模型都来自`three.js`，版权归原作者所有。

由于该项目没有服务端，所以模型和场景的保存载入功能无法使用。

## 演示地址

https://tengge1.github.io/ShadowEditor-examples/demo/index.html
